# Validate country input

Validate country input

## Usage

``` r
validate_country_input(x)
```

## Arguments

- x:

  An object to be validated as suitable as input for the `country`
  argument of
  [`daedalus()`](https://jameel-institute.github.io/daedalus/reference/daedalus.md).

## Value

A `<daedalus_country>` object.
