# Make a hospital overflow response

Intended to be a helper function for internal use only. No parameters
are carried although this may change in future.

## Usage

``` r
new_daedalus_hosp_overflow(country)
```

## Arguments

- country:

  A `<daedalus_country>`.

## Value

An object of class `<daedalus_hosp_overflow>` that inherits from
`<daedalus_response>`.
