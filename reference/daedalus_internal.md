# Internal function for daedalus

Internal function for daedalus

## Usage

``` r
daedalus_internal(
  time_end,
  parameters,
  initial_state,
  flags,
  ode_control,
  n_groups
)
```

## Value

A list of state values as returned by
[`dust2::dust_unpack_state()`](https://mrc-ide.github.io/dust2/reference/dust_unpack.html).
