# Validate and return infection input

Validate and return infection input

## Usage

``` r
validate_infection_input(x)

validate_infection_list_input(x)
```

## Arguments

- x:

  An object to be validated as input to the `infection` argument of
  [`daedalus()`](https://jameel-institute.github.io/daedalus/reference/daedalus.md).

## Value

`validate_infection_input()` returns a `<daedalus_infection>`.
`validate_infection_list_input()` returns a list of
`<daedalus_infection>`s.
