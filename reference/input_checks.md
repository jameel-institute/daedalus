# Check that a class has expected names

Check that a class has expected names

## Usage

``` r
assert_class_fields(x, expected_fields)
```

## Arguments

- x:

  An object to be tested as having `expected_fields` class member names.

- expected_fields:

  A character vector of expected class member names.

## Value

Invisibly returns `x`; primarily called for its side effects.
