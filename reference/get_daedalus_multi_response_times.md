# Get response times from a dust2 system with multiple groups

Get response times from a dust2 system with multiple groups

## Usage

``` r
get_daedalus_multi_response_times(output, n_groups)
```

## Arguments

- output:

  dust2 output
  [`daedalus_internal()`](https://jameel-institute.github.io/daedalus/reference/daedalus_internal.md).
