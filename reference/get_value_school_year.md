# Calculate the value of a school year

Calculate the value of a school year

## Usage

``` r
get_value_school_year(gni)
```

## Arguments

- gni:

  The GNI per capita of a country. Must be a single value. See
  [daedalus.data::country_gni](https://jameel-institute.github.io/daedalus.data/reference/life_value.html)
  for values.

## Value

A single value giving the value of a school year given the country GNI
per capita and an expected work period of 45 years.
